# 📜 Código de Conducta

## Nuestro compromiso

Este es un espacio donde todo el mundo es bienvenido, independientemente de
nivel de experiencia, género, orientación, discapacidad, apariencia, raza o religión.

## ✅ Comportamiento esperado

- Usar lenguaje amable e inclusivo
- Respetar puntos de vista diferentes
- Aceptar críticas constructivas con buena actitud
- Centrarse en lo que es mejor para la comunidad
- Mostrar empatía hacia otros miembros

## ❌ Comportamiento inaceptable

- Lenguaje o imágenes sexualizadas
- Trolling, insultos o ataques personales
- Acoso público o privado
- Publicar información privada de otros sin permiso
- Cualquier conducta que se considere inapropiada

## 📩 Reporte

Si ves algún comportamiento que viola este código, repórtalo contactando
directamente a [@ROKO108](https://github.com/ROKO108).

Todos los reportes serán revisados y tratados con confidencialidad.
